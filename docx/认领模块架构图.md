### 各层详细说明

### 示例 ： 认领模块

#### 1. Controller 层 (ClaimController)

**文件位置**: `claim/ClaimController.java`

- 负责接收 HTTP 请求
- 使用 `@Valid` 注解进行参数校验
- 调用 Service 层处理业务逻辑
- 返回统一的 `ApiResponse` 响应格式

| 方法 | 路径 | 功能 | |------|------|------| | POST | `/api/claims` | 创建认领申请 | | GET | `/api/users/me/claims` | 查询我的认领记录 |

------

#### 2. Service 接口层 (ClaimService)

**文件位置**: `claim/service/ClaimService.java`

定义了认领模块的业务接口，封装业务流程：

```
public interface ClaimService {
    void createClaim(ClaimDTO.CreateClaimReq request);
    List<ClaimDTO.ClaimResp> getMyClaims();
}
```

------

#### 3. Service 实现层 (ClaimServiceImpl)

**文件位置**: `claim/service/Impl/ClaimServiceImpl.java`

实现具体的业务逻辑：

| 业务方法 | 核心逻辑 | |---------|---------| | **createClaim()** | ① 验证当前用户已登录 ② 验证物品存在 ③ 验证物品是招领(scene="found") ④ 验证不能认领自己发布的物品 ⑤ 验证没有重复提交(REVIEW_PENDING/APPROVED状态) ⑥ 创建认领记录，生成bizId(CLM+UUID) ⑦ 设置状态为REVIEW_PENDING ⑧ 插入数据库 | | **getMyClaims()** | ① 验证当前用户已登录 ② 查询当前用户的所有认领记录 ③ 按创建时间倒序 ④ 关联查询物品信息和用户信息 ⑤ 格式化创建时间 ⑥ 返回DTO列表 |

------

#### 4. DTO 层 (ClaimDTO)

**文件位置**: `claim/dto/ClaimDTO.java`

数据传输对象，用于请求和响应：

| 类名 | 用途 | |------|------| | **CreateClaimReq** | 创建认领请求: itemId, proofDescription, contact | | **ClaimResp** | 认领响应: id, bizId, itemId, itemName, claimantId, claimantName, proofDescription, contact, status, auditRemark, createTime | | **ClaimQueryReq** | 查询请求: pageNum, pageSize, status, itemId |

------

#### 5. Mapper 层

**文件位置**: `mapper/ClaimMapper.java`、`mapper/ItemMapper.java`、`mapper/UserMapper.java`

继承 MyBatis-Plus 的 `BaseMapper`，提供数据库操作：

| Mapper | 主要方法 | |--------|----------| | **ClaimMapper** | insert(), selectList(), exists() | | **ItemMapper** | selectById() | | **UserMapper** | selectById() |

------

#### 6. 实体层 (Claim)

**文件位置**: `domain/entity/Claim.java`

数据库表对应的实体类：

| 字段 | 说明 | |------|------| | id | 主键ID | | bizId | 业务ID(CLM+UUID) | | itemId | 物品ID | | claimantId | 认领人ID | | draftId | 草稿ID | | formalProof | 正式证明材料 | | contact | 联系信息 | | status | 状态(REVIEW_PENDING/APPROVED/REJECTED/EXPIRED/CANCELLED) | | auditRemark | 审核备注 | | createdAt | 创建时间 |

------

#### 7. 工具类

| 工具类 | 位置 | 功能 | |--------|------|------| | **SecurityUserUtils** | `security/SecurityUserUtils.java` | 获取当前登录用户ID |
