---

### 3.3 修复 NotificationToast Vue Warning

**文件**：`frontend/src/components/common/NotificationToast.vue`

**问题**：控制台 `[Vue warn]: Invalid prop: custom validator check failed for prop "type"`

**修复**：在 props 中为 `type` 添加 validator

```javascript
props: {
  type: {
    type: String,
    default: 'info',
    validator: (v) => ['info', 'success', 'error', 'warning'].includes(v)
  }
}
```

