# 后端架构

Spring Boot 架构： Spring Boot 3.3.5 + MyBatis-Plus + MySQL + Spring Security + JWT + Spring Boot Mail

## 目录结构

```
src/
├── main/
│   ├── java/
│   │   └── com/
│   │       └── campus/
│   │           └── lostfound/
│   │               ├── LostFoundApplication.java # 应用启动类
│   │               ├── admin/ # 管理员模块
│   │               ├── auth/ # 认证模块
│   │               ├── catalog/ # 商品目录模块
│   │               ├── claim/ # 认领模块
│   │               ├── common/ # 通用模块
│   │               ├── config/ # 配置模块
│   │               ├── domain/ # 领域模型
│   │               ├── found/ # 招领模块
│   │               ├── lost/ # 失物模块
│   │               ├── mapper/ # 数据访问层
│   │               ├── notice/ # 公告模块
│   │               ├── report/ # 举报模块
│   │               ├── security/ # 安全模块
│   │               └── user/ # 用户模块
│   └── resources/
│       ├── application.yml  # 应用配置文件
│       ├── static/  # 静态资源目录
│       └── templates/  # 模板目录
└── test/
    └── java/
        └── com/
            └── campus/
```

（每个模块有 dto + service\&Impl + controller）

## 关键决策

- 使用 Spring Boot 3.3.5 作为基础框架
- 使用 MyBatis-Plus 作为持久层框架
- 使用 MySQL 作为数据库
- 使用 Spring Security + JWT 实现安全认证
- 使用 BCrypt 进行密码加密
- 使用 Spring Boot Mail 实现邮件服务
- 使用 Lombok 减少样板代码

## 技术栈

- 核心框架: Spring Boot 3.3.5
- 持久层: MyBatis-Plus 3.5.7
- 数据库: MySQL
- 安全: Spring Security + JWT (Json Web Token)
- 密码加密: BCrypt
- 邮件服务: Spring Boot Mail
- 依赖管理: Maven
- 编程语言: Java 17

## 开发命令

```bash
mvn clean install
mvn spring-boot:run
mvn clean package
```
