# 用户模块 - 注册接口及邮箱验证码更新说明

## 1. 邮箱验证码接口

**请求路径**: `/api/auth/send-email-code`
**请求方式**: `POST`
**描述**: 为用户注册获取邮箱验证码

### 请求参数 (JSON)

| 字段 | 类型 | 必填 | 描述 |
| --- | --- | --- | --- |
| email | String | 是 | 用户的邮箱地址，格式必须为有效的邮箱格式 |

### 响应参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | Integer | 状态码，0 表示成功 |
| message | String | 提示信息 |
| data | Object | 响应数据，成功时为 null |

---

## 2. 用户注册接口 (更新)

**请求路径**: `/api/auth/register`
**请求方式**: `POST`
**描述**: 用户注册，新增邮箱及验证码字段

### 请求参数 (JSON)

| 字段 | 类型 | 必填 | 描述 |
| --- | --- | --- | --- |
| studentNo | String | 是 | 学号 |
| name | String | 是 | 姓名 |
| password | String | 是 | 密码 |
| phone | String | 否 | 手机号 |
| email | String | 是 | 邮箱 |
| emailCode | String | 是 | 邮箱验证码 (新增) |

### 响应参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| code | Integer | 状态码，0 表示成功 |
| message | String | 提示信息 |
| data | Object | 响应数据，成功时为 null |
