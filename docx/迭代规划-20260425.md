## 四、API 路径一致性检查

### 前后端 API 映射

| 功能 | 前端 API | 后端路径 | 状态 |
|------|----------|----------|------|
| 物品列表 | `itemApi.getItems(scene)` | `GET /api/items` | ✅ |
| 物品详情 | `itemApi.getItemDetail(id)` | `GET /api/items/{id}` | ✅ |
| 发布物品 | `itemApi.createItem(data)` | `POST /api/items` | ✅ |
| 我的发布 | `userApi.getMyItems()` | `GET /api/items/me` | ✅ |
| 物品审核 | `adminApi.getReviewQueue()` | `GET /api/admin/items/review` | ✅ |
| 审核操作 | `adminApi.auditItem()` | `POST /api/admin/items/audit` | ✅ |
| 联系方式解锁 | `itemApi.unlockContact()` | `GET /api/items/{id}/contact` | ✅ |
| 举报提交 | `reportApi.createReport()` | `POST /api/reports` | ✅ |
| 举报列表 | `reportApi.getMyReports()` | `GET /api/reports` | ✅ |
| WebSocket通知 | `notificationStore.initWebSocket()` | STOMP over WebSocket | ✅ |
