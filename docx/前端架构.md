# 前端架构

Vue 3 架构： Vite + JavaScript + Pinia + Vue Router + Axios

│ ├── api/ # 所有接口封装（最重要！）
│ │ ├── index.js # axios 实例
│ │ ├── modules/ # 按业务模块拆分接口
│ │ │   ├── admin.js
│ │ │   ├── auth.js
│ │ │   ├── categories.js
│ │ │   ├── claims.js
│ │ │   ├── index.js
│ │ │   ├── items.js
│ │ │   ├── notices.js

│ │ └── interceptors.js # 请求/响应拦截器
│ ├── assets/ # 图片、svg、字体等
│ ├── components/ # 全局公共组件
│ │ ├── common/ # 通用组件（Button、Table、Modal 等）
│ │ └── business/ # 业务组件（可复用）
│ ├── composables/ # 组合式 API 逻辑复用（useRequest、useAuth 等）

## 关键决策

- `axios` 位于 `src/api/` 目录，具有集中式拦截器
- 业务 API 调用分组在 `src/api/modules/` 下
- `Pinia` 在 `src/stores/` 中初始化，用户认证状态在 `src/stores/modules/user.js` 中
- 路由定义按领域划分在 `src/router/modules/` 下
- 认证和权限控制在 `src/router/guards.js` 中实现
- 可复用布局模板在 `src/layouts/` 中
- 通用 UI 组件和业务 UI 组件分别在 `src/components/common` 和 `src/components/business` 下

## 技术栈

- 构建工具: Vite
- 编程语言: JavaScript
- 状态管理: Pinia
- 路由管理: Vue Router
- HTTP 客户端: Axios
- UI 框架: 原生 Vue 组件

## 开发命令

```bash
npm install
npm run dev
npm run build
```
